# Hardware Notes (Acer E5‑571)

This section documents the hardware characteristics, limitations, and considerations that shaped the rebuild process.

## System Overview
- Model: Acer Aspire E5‑571‑3952
- Age: Legacy hardware (pre‑TPM era)
- CPU: Intel (older generation)
- RAM: Limited but functional for lightweight Linux distributions
- Storage: Sufficient for a clean Kali Linux installation

## Key Limitations
- No TPM chip (preventing Windows 11 installation)
- Legacy BIOS behaviour
- USB boot inconsistencies
- Locked Windows 10 account with no recovery options
- Wi‑Fi chipset requiring additional troubleshooting

## Why Kali Linux Was Chosen
- Lightweight enough for older hardware
- No TPM requirement
- Strong support for cybersecurity tools
- Ideal for hands‑on learning and practical IT support development

## Practical Takeaways
- Older hardware can still be repurposed effectively
- Lack of TPM can be an advantage for open‑source experimentation
- Hardware constraints encourage creative problem‑solving
