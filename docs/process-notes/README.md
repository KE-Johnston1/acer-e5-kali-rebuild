# Rebuild Process Notes

This section documents the practical steps, decisions, challenges, and lessons learned during the Acer E5‑571 rebuild project.

It includes:

- Why the rebuild was necessary
- Limitations of the hardware (no TPM, legacy BIOS behaviour)
- Why Windows 11 was not an option
- Why Kali Linux was chosen
- How AI tools and online research supported the process
- Time taken (approx. 5 hours + 2 hours planning)
- Skills demonstrated during the rebuild
